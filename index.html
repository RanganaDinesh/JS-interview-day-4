2.What are pure functions?

        A pure function in JavaScript is a function that returns the same result if the same arguments(input) are passed in the function
        exampls:
        function operationAdd(a, b){ // A pure function adding two integers passed in it.
            return a+b;
        }
        operationAdd(2,3)//output :- 5

3.What is callback hell

        Callback Hell is a pattern with multiple nested callbacks which makes code hard to read and debug when dealing with asynchronous logic.
         The callback hell looks like a pyramid structure

         example:
         function step(callback) {
            console.log("data")
            callback()
            
          }
          function step1(callback) {
            console.log("data1")
            callback()
            
          }
          function step2(callback) {
            console.log("data2")
            callback()
            
          }
          function step3(callback) {
            console.log("data3")
          callback()
            
          }
          step(function(){
            step1(function(){
              step2(function(){
               step3(function(){
                 console.log("result")
               })
              })
            })
          })

7.What is a Temporal Dead Zone?

        Temporal Dead Zone is the period of time during which the let and const declarations cannot be accessed. 
        Temporal Dead Zone starts when the code execution enters the block which contains the let or const 
        declaration and continues until the declaration has executed
